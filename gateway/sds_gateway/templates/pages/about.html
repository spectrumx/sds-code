About the SpectrumX Data System

The SpectrumX Data System (SDS) is a data management and retrieval platform for radio-frequency experiments. This is the SDS Gateway, a web interface for user-friendly data retrieval. Here you can generate API keys to interact with SDS via our Python SDK, explore datasets, and manage your radio-frequency experiment data.

The SpectrumX Data System (SDS) is a project under the SpectrumX initiative, which vision is to become a trusted resource within the spectrum ecosystem offering objective, long-term and innovative policy and technical contributions through collaborative, inclusive and integrative education and research activities.
